### 완료(주요)
- 디자인 시스템(AppColors/AppTypography/AppSpacing) + 공통 위젯(Button/Input/Card/Loading)
- 인증 UI(스플래시/로그인/회원가입) + 오류 DETAILS 안전 처리
- 카카오 로그인(Android 복귀/키 주입) + 카카오 버튼(가이드 스타일, 심볼 PNG만 추가 필요)
- 하단 네비(Figma-style) + 3탭(홈/쿠폰함/친구목록)
- 알림 설정 화면(NotificationSettingsScreen)
- 친구 수락 대기 화면(`/friends/requests`) + 친구목록 브랜디아 UI(검색바 1개 + 카드 리스트)
- 홈 UI 1차(배경 `#FAFAFA`, 상단 레이아웃/성공한날/오늘의 걸음 수 카드 “픽셀-ish”)

### 다음 우선순위(기능)
1. **걸음 수 실측 연동(권한/수집/동기화)**: 미션/통계/친구 일일걸음수의 기반 데이터
2. 회차(10일) “서버 기준” 시작일/종료일 확정 + 홈 상단 기간 표시 고정
3. 내 정보 보기(통계) 화면(UI) + 회차 누적(총 걸음/거리/kcal) + kcal→음식(상위 1개)
