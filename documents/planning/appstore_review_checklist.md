# App Store 심사 체크리스트 (MVP)

> 목적: 심사 리젝 포인트를 빠르게 제거하기 위한 내부 체크리스트입니다.

## 진행 현황

- [x] 계정 삭제 경로 제공 (외부 링크): 설정 → `계정 삭제 요청(탈퇴)` → `https://doyakmin.com/delete-account`
- [ ] iOS “Sign in with Apple” 제공 (구글/카카오 등 제3자 로그인 사용 시 필수)
- [x] iOS 권한 문구(Info.plist): 모션/위치 사용 설명 문구 점검
- [ ] 리뷰 노트 준비: 테스트 계정/권한 안내/핵심 플로우 요약

## 남은 항목 수

- 총 4개 중 완료 2개 / 남음 2개
