# 남구이야기 PRD v1.0

**프로젝트**: 남구이야기 (Namgu Story)  
**버전**: 1.0  
**작성일**: 2024-12-22  
**기술 기준**: Flutter + Firebase

---

## 1. 프로젝트 개요
- **앱 목적**: 부산 남구 지역 기반 걷기 미션/쿠폰 앱
- **핵심 가치**: 걷기 → 미션 완료 → 쿠폰 제공 → 지역 상권 연계
- **대상 사용자**: 30~40대 여성, 부산 남구

## 2. 핵심 기능
1. **쿠폰 시스템**
   - 쿠폰 발급/사용, 4자리 코드 확인
2. **미션 시스템**
   - 걸음수/방문/초대 등 미션
3. **지도**
   - 가맹점 표시 및 정보 확인
4. **알림**
   - 쿠폰/미션/공지 푸시 알림
5. **인증**
   - 이메일/비밀번호 + 카카오 로그인

## 3. 성공 지표
- 회원가입 전환율
- 미션 완료율
- 쿠폰 사용 전환율
- 재방문률(D7/D30)

## 4. 범위
- **MVP 1차**: 인증, 쿠폰, 미션(걸음수), 지도, 알림
- **MVP 2차**: 친구 초대, 룰렛, 구독
